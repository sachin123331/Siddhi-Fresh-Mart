## 🎯 **How to Use**

### **For Customers**
1. Click Login
2. Select "Customer" role
3. Enter your name and phone
4. Browse products and add to cart
5. Enter delivery address
6. Select location and payment method
7. Place order
8. Rider will contact you

### **For Admin (Sachin)**
1. Click Login
2. Select "Admin" role
3. Enter "Sachin" as name
4. Go to Admin Panel tabs:
   - **Riders:** Add/manage riders
   - **Shopkeepers:** Add/manage shopkeepers
   - **Orders:** View and manage orders
   - **Payments:** View earnings and process weekly payment
